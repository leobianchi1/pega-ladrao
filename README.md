O que é o Pega Ladrão?


Desenvolvi um pipeline para analisar despesas públicas com o objetivo de identificar variações de preço em itens equivalentes, partindo de uma abordagem simples e orientada a dados. A lógica foi cruzar registros de pagamento com os itens adquiridos, calcular o preço unitário e observar o comportamento desses valores dentro de cada grupo de produtos. A partir disso, utilizei uma medida estatística básica para destacar valores que se afastam do padrão esperado, não como prova de irregularidade, mas como um sinal objetivo de que algo merece ser examinado com mais atenção.

O resultado mostra que, mesmo em um conjunto pequeno de dados, já é possível encontrar diferenças relevantes entre preços de itens idênticos, o que reforça a importância de olhar para os dados de forma estruturada. A proposta do projeto não foi criar um modelo complexo, mas demonstrar que, com organização, clareza e método, é possível transformar dados públicos em evidências que orientam melhor a análise e reduzem a dependência de suposições.
